캐나다 핀치마트 인스타용 **카드뉴스 4장 + 캡션**을 만든다. 입력: $ARGUMENTS

**마음가짐: 초짜가 아니라 시니어 SNS 마케터** ([[feedback_senior_copywriting_mindset]]) — 스펙 나열이 아니라
스크롤 멈추는 후킹·고객심리·전환까지 책임진다. 단 **§9 데이터 진실성**(검증된 라벨/스토어 정보만
매력적으로 프레이밍, 날조 금지)은 절대 유지.

## 절차

1. **제품 식별·사실 수집:** `output/new-item/<slug>/` 의 `*_product_info.json` + `*_detail.html` 를 읽어
   검증된 특징·향/맛·용량옵션·USP를 뽑는다. 옵션(사이즈/맛) 여러 개면 **다 표기**.

2. **대표 이미지 선정 (시니어 체크):** 커버는 첫인상이 전환의 절반. **흰 배경 클린 제품컷** 우선.
   매대 진열컷(손·선반)만 있으면 "광고 소재로 약하다" 1줄 플래그하고 클린컷 요청하되, 없으면 있는 걸로 진행.
   (rep_image / 사용자 제공 URL / 로컬 경로 모두 가능 — 스크립트가 URL·경로 둘 다 처리.)

3. **시니어 카피 작성:**
   - **커버 hook:** 제품명·스펙 ❌ → **고객 페인포인트/베네핏**으로 (예: "한 스쿱이면, 빨래가 온종일 좋은 향").
     `sub`(한 줄 설명)·`title`(제품명)·`scent`(향/맛)·`sizes`(용량옵션 전부).
   - **features 3개** (개수 균형): `{icon, title, body}`. **쉬운 한글**(영어 음역 듀얼센트·올데이프레시니스 ❌
     → "두 가지 향을 한 알에"·"하루 종일 가는 향"). body 핵심은 `<strong style='color:#e0483f;'>` 강조.
   - **recommendations 3개:** "이런 분들께" — 시나리오·타겟. 핵심 `<strong style='color:#e0483f;'>` 강조.
   - **cta:** `{hook(베네핏 카피), search:"캐나다 핀치마트"}`.
   - **caption:** ① **1번째 줄=후킹**(인스타는 첫 1~2줄만 노출) ② 공감 스토리 ③ ✅ 베네핏 불릿
     ④ **저장 유도**(💬) ⑤ CTA "프로필 링크"([[reference_sns_bio_link]] Linktree) + "네이버 검색"
     ⑥ 해시태그 ~20개 전략 배분(브랜드/카테고리/라이프스타일/시나리오/직구/롱테일).

4. **🔑 원산지 오해 회피:** 제품 원산지가 **캐나다가 아니면**(다우니=미국, 허쉬=미국 등) 카드·캡션에
   `캐나다 직수입`/`캐나다산` 금지(소비자가 캐나다 제조로 오해). `해외직구`·`미국직구`·실제 원산지로.
   원산지가 실제 캐나다인 제품(PC 메이플시럽·RW가르시아 등)만 `캐나다 직수입` OK. [[feedback_origin_vs_release_market]]

5. **spec JSON 작성 → 스크립트 실행:**
   ```
   python3 scripts/build_sns_cards.py <spec.json>
   ```
   레이아웃·디자인 토큰(공지 스타일: 코랄레드 #e0483f + 크림핑크 #fff7f4, 잘난체+나눔스퀘어라운드,
   둥근/점선 카드, **CTA 네이버 초록창**, **푸터 로고만**, 커버 **넘김 유도 👉**)은 스크립트가 소유.
   spec 스키마·예시는 `scripts/build_sns_cards.py` 상단 docstring 참고. 출력: `output/sns/<slug>/cardN.png`.

6. **전달:** PNG 4장을 SendUserFile 로 보내고(캐러셀 순서 커버→특징→추천→CTA), 캡션 전문을 보여준다.
   필요 시 카드별로 다시 렌더(문구·간격 수정 후 `build_sns_cards.py` 재실행).

## 디자인 규칙 (확정)
- 1080×1080 4장: **커버(후킹+넘김유도) → 제품특징 3 → 추천 3 → CTA(네이버 초록창)**
- 공지 디자인 일관, 쉬운 한글, 영어 음역 자제, 용량옵션 다 표기, 푸터=로고만
- 커버 제품컷은 클린컷 우선, 매대컷은 플래그
- 캡션 URL은 클릭 안 됨 → "프로필 링크(Linktree)"·"네이버 검색" 유도 [[reference_sns_bio_link]]

## 모드 2 — MZ 라인업 (포토덤프, croket 스타일) — `mz`

판매중 한 **라인업(그룹상품)** 을 4:5(1080×1350) 실사진 카드뉴스로. ([[project_sns_mz_croket_style]])
스크립트: **`python3 scripts/build_sns_mz.py [그룹ID] [출력폴더명]`** (기본 다우니 언스토퍼블 50533349).

**🔑 역할 파이프라인 (MZ → 시니어 마케터):**
1. **MZ** = `guide/Product_*.csv` 에서 그룹ID로 제품 추출(사이즈 dedupe, 대표=최대용량컷) → 향별 포토
   쇼케이스 카드(실사진+마커 헤드라인+말풍선+제품 메인컬러 테마). `build_sns_mz.py` 의 `COPY` 가 content layer.
2. **시니어 마케터** = MZ가 채운 향 카피를 **공식 페이지(downy.com 등)로 팩트체크**(절차: market-researcher 에이전트로
   라인업 전 제품 공식 향 설명 수집 → COPY 교정). 그 뒤 **💝 향 추천 가이드(페르소나→향) + 🛒 마무리 CTA(네이버
   초록창) 카드** 가 자동 붙고, **캡션** 작성. 팩트체크에서 향·이미지·컬러 문제 발견 시 **MZ(COPY/이미지) 재작업**.
3. 커버엔 **"※ 향 종류는 예고 없이 추가·단종될 수 있어요"** 디스클레이머 포함(스크립트 내장).

**MZ 모드 사실성 규칙(공식 계정):** [[feedback_senior_copywriting_mindset]] — 향 노트·무드는 공식 검증분만,
**팩 공통 슬로건("crafted by world-class perfumers" 등) 번역 금지·vague 타이틀 금지**, 제품 메인컬러·대표컷은
**이미지 열어 확인**. 원산지≠캐나다면 캐나다직수입 배지 금지.
