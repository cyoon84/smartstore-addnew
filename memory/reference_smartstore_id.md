**Store ID:** `finchmart_ca`

**상품 URL 패턴:**
`https://smartstore.naver.com/finchmart_ca/products/{product_id}`

예시:
- 다우니 2in1 씨사이드 스파 (그룹 옵션): https://smartstore.naver.com/finchmart_ca/products/13529014916

**활용:** product_info.json에 `live_url`, `live_product_id` 필드로 등록 완료 후 기록. 추후 매출/노출 추적이나 동일 상품 비교에 사용.
