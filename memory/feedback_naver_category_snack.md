**네이버 스마트스토어 식품 카테고리 — 검증된 트리 (2026-06-01 셀러 화면 확인).**

스낵·과자류 상위는 **`식품 > 스낵/과자`** 다. ~~`식품 > 과자/베이커리`~~ 라는 상위는 **없다**(빵류는 `식품 > 빵/베이커리` 별도).
`스낵/과자` 의 **세분류(잎 노드)는 이게 전부**:
> 가공안주류 · 과자/쿠키세트 · 기타과자 · 시리얼 · 일반스낵 · 쿠키/비스킷 · 파이/케익 · 팝콘/강냉이류

**🚫 지어내기 금지:** `감자스낵`·`감자칩`·`포테이토칩`·`콘스낵`·`나초/토르티야칩`·`크래커`·`조각케이크`·`스낵/팝콘`·`스낵/시리얼`·`스낵/칩`·`스낵/간식거리` 같은 잎/상위 노드는 **존재하지 않는다.** listing-writer 가 그럴듯하게 만들어내던 것 — 화면에 없으면 강제이동·판매중지 경고.

**매핑 (이 세분류 안에서만 고른다):**
- 감자칩·감자스틱·콘스낵(치즈퍼프)·토르티야/나초칩·크래커 등 짭짤한 스낵 → **일반스낵**
- 팝콘 → **팝콘/강냉이류**
- 쿠키·비스킷 → **쿠키/비스킷**
- 파이·케이크(스낵케이크 포함) → **파이/케익**
- 시리얼·그래놀라 → **시리얼**
- 어소트/선물세트 → **과자/쿠키세트**
- 마른안주·가공안주 → **가공안주류**
- 위에 안 맞으면 → **기타과자**

**초콜릿·커피·차·캔디 등은 이 세분류에 없다** → 다른 상위(예: 초콜릿류, 식품>커피, 식품>차)로 가야 하며, 정확 노드는 **등록 화면에서 확인**. 확실치 않으면 잎 노드를 지어내지 말고 검증된 노드 중 가장 가까운 것 + "(화면 확인)" 표기. 최종 선택은 셀러가 등록 화면에서 한다.
