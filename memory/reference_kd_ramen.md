**KD 라면 (KD Ramen / Cheesy Ramen Bowl)** — 2026-07-10 등록.

**🔑 브랜드 vs 제조사 (반복 오류 — 정확히):**
- **브랜드 = KD (크래프트 디너 / Kraft Dinner)** — 캐나다 국민 맥앤치즈 브랜드. "크래프트 디너"(디너, NOT 다이너).
- **제조사 = 크래프트 하인즈 (Kraft Heinz)** — 크래프트 디너를 만든 회사. KD 라면을 새로 출시.
- 카피 정형: **"크래프트 디너를 만든 크래프트 하인즈에서 새로 출시한 KD 라면."** ("KD/크래프트 디너가 만든" ❌ — 브랜드는 만든 주체가 아님.)

**제품:** 용기면(볼), 맛/매운맛 3종 — 마일드(엑스트라 치즈, 97g) / 미디엄(치즈, 95g) / 엑스트라 스파이시(핫, 97g). 진한 체다 치즈 베이스. 알레르기 우유·밀. 팩 전면 NEW/NOUVEAU 배지(신제품 사실 근거). 원산지 캐나다.

**등록:** 월마트 원가 $3.47, 세금 0%(기본식품), 마진 +$2, FX1083 → **판매가 ₩6,350**(3종 동일). 배송 **4개당 15,000**(수량별 AP=4). 카테고리 **식품>라면/면류>라면>컵라면 (50014000)**. **맛별 개별 단일상품 3개**(그룹/멀티옵션 아님 — 사용자가 옵션은 직접 구성, [[feedback_group_conversion_workflow]]). 슬러그: kd_ramen_cheese_medium_95g / kd_ramen_extra_cheese_mild_97g / kd_ramen_extra_spicy_hot_97g. 공식페이지 이미지(cdn.allotta.io — 서버 200, 네이버 생존 가능)를 상세에 활용.

**🔑 라면/컵라면 태그 축 첫 사전 (입력화면 확정 10개, 검색량):** `치즈라면`(708,251) · `용기면`(954,131) · `수입라면`(552,927) · `자취요리`(310,877) · `치즈맛`(284,779) · `매운라면`(205,437) · `캐나다직수입`(130,204) · `캠핑음식`(89,770) · `전자렌지조리`(17,777) · `캐나다산`(16,189). (미채택: 국내미발매·외국라면·별미라면·야식추천·홈파티음식·크래프트디너.)

**🔑 상품명 = 검색어 앞세우기:** 브랜드(KD) 국내 인지도 0이라 상품명 앞에 두면 유입 X → **`치즈라면`을 앞으로**, 브랜드는 뒤로. 예: `치즈라면 크래프트디너 KD 미디엄 95g`. (신제품 브랜드는 상품명 맨앞 금지, 카테고리·속성어 우선 — §5·§8.) [[feedback_brand_country_vs_made_in]]
