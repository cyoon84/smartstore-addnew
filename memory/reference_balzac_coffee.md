**발작 커피 = 우리가 파는 캐나다 스페셜티 원두 라인.** 발작 관련 상세·태그·가격 작업 시 이 팩트를 소스로 쓴다(공홈 balzacs.com + 코스트코 + 실사진 확인분).

**브랜드 스토리 (공홈 공식, 검증됨):**
- Balzac's Coffee Roasters = 캐나다 **온타리오** 스페셜티 로스터리. 프랑스 대문호 **오노레 드 발자크**(하루 50잔 마시며 집필) 이름을 땀.
- **파리 그랑 카페** 문화를 캐나다에 옮기려는 열정에서 시작 → **1993년 커피 카트** → **1996년 온타리오 스트랫퍼드(셰익스피어 마을과 동명) 1호점** → 현재 **온타리오 16개 매장** + 캐나다 마트 유통. 봉지 EST 1993.
- 명언 "커피는 내 삶의 위대한 힘이다"(Coffee is a great power in my life) — 봉지 측면.
- 🔑 **"캐나다 전국 마트" 과장 금지** (2026-07-09 사용자 지적 — 캘거리 코스트코엔 안 팖). 상세 카피는 "온타리오 사람들이 오래도록 사랑해온 스페셜티 로스터리" 톤. 원산지=캐나다 온타리오 로스팅(생두는 수입 아라비카).

**소싱·가격 (2026-07-09 확정):**
- 340g 전부(파머스·발작스·다크어페어·디카페인) = 로블로/코스트코 **$16.99** → 판매 **₩22,000**(배송 ₩10,000, 마진~$2). 디카페인 340g에 가격 맞춤.
- 파머스 907g = 코스트코 **$23.99** → 판매 **₩27,900**(배송 ₩15,000). 노출가 낮추고 마진 배송이동(§0-D·§7-2, 총 +₩4,787/개).
- 디카페인 907g = **공홈 $49**(제품$39+배송$10) → ₩62,900 / 2.27kg = 공홈 $109 → ₩138,000(배송 ₩17,000). 공홈 전용(코스트코·로블로 미취급).
