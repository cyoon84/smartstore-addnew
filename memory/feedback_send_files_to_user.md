원격 세션(remote session)에서 생성한 **사용자용 산출물 파일**(발송처리 엑셀, 업로드용 엑셀, 정산 md, 일괄등록 엑셀 등)은 로컬 경로만 알려주지 말고 **SendUserFile 도구로 채팅에 첨부**한다 — 사용자가 모바일 앱이나 다른 컴퓨터에서 대화를 열어 바로 다운로드할 수 있게.

**Why:** 파일이 mac mini 로컬에만 있으면 사용자가 다른 기기에서 작업할 때(폰에서 스마트스토어 업로드 등) 접근할 수 없다. 채팅 첨부는 기기 무관.

**How to apply:** 워크플로 산출물이 완성되는 시점(한미플로우 발송처리 파일, 일괄등록 엑셀 등)에 `SendUserFile`(display: attach)로 첨부. 중간 작업 파일·임시 파일은 제외 — 사용자가 실제로 다른 데서 쓸 최종 파일만. (2026-07-14 사용자 지시: "remote-session으로 만든 파일을 다른 컴퓨터나 모바일에서 볼 수 있게 해줘… 지금은 필요없고 앞으로")
