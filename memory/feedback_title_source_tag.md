스마트스토어 상품명 끝의 출처 태그는 다음 우선순위로 결정. **"정품"이라는 단어는 광고 클리셰라 사용하지 않음.**

> **🔴 2026-05-31 핵심 갱신 — 상품명에 국가(캐나다/미국)를 넣지 않는다.** 네이버는 **원산지 필드** 값을 상품명 끝에 **자동으로 `[원산지:캐나다]`** 형태로 붙인다(스마트스토어 노출 화면 확인). 그래서 상품명 텍스트에 "캐나다"를 또 넣으면 **이중 표시(중복)**가 된다. → **국가 출처는 원산지 필드로만 설정**하고 **상품명에선 뺀다.** (기존 "일반 유통은 '캐나다' 단독 표기" 룰은 이걸로 superseded.) 매장 한정 SKU의 **매장명(코스트코 등)**은 네이버가 자동 표시하지 않으므로 필요 시 상품명에 유지하되, **"캐나다"는 빼고 "코스트코"만** (국가는 원산지 필드가 처리 → "…코스트코 [원산지:캐나다]"로 노출). 2026-05-31 팀홀튼 핫초코 케이스에서 사용자 명시("캐나다 빼자, 원산지 캐나다로 이미 나와있어서").

> **2026-05-24 확정 — 네이버 SEO 가이드와의 충돌 해소:** 네이버 공식 쇼핑검색 SEO 가이드는 상품명에 판매처명(매장명) 기입을 금지하나, 사용자가 **"캐나다 코스트코는 유지하자"**고 명시 결정. 매장 한정 SKU의 "캐나다 {매장명}" 출처 태그는 **본 룰대로 계속 유지**한다 (SEO 페널티 가능성은 사용자가 인지·감수). [[naver-title-seo-guide]] 의 다른 규칙(동의어 중복 금지·50자 내외·홍보성 단어 금지·브랜드 한글 표기 등)은 그대로 적용.

## 우선순위

1. **매장 한정 SKU**: 그 매장에서만 파는 PB·콜라보·한정 구성이면 **"캐나다 {매장명}"** (예: "캐나다 코스트코" — Kirkland Signature 등 코스트코 한정 SKU). 매장명이 단독 검색어로도 트래픽이 큼.
2. **일반 유통 정품**: 여러 매장(Walmart·Loblaws·Amazon 등)에서 동일 SKU로 파는 일반 유통이면 **"캐나다"** 단독 표기. **"정품"이라는 단어는 붙이지 않음.**
3. **출처 표기 자체 생략**: 50자 제약으로 빠듯하거나 출처가 셀링 포인트가 아닐 때.

## Why

- "캐나다 코스트코"는 매장 한정 SKU 신뢰감 + "코스트코" 검색 트래픽 동시에 잡음.
- "정품" 단어는 어디서나 흔히 쓰여 차별화 안 되고, 광고 클리셰처럼 보임. 사용자가 2026-05-16 "정품도 빼자"고 명시해 제거 결정.
- 매장 한정 아닌 일반 유통에 "캐나다 코스트코"를 붙이면 허위 표시 — 매장이 확실할 때만 매장명 사용.

## How to apply

- 사용자 메시지에서 (1)매장 한정 SKU인지 (2)일반 유통인지 한 번 확인. 사진 상 Costco 매대지만 일반 유통 SKU일 수 있으니 사용자에게 검증.
- 매장 한정 → "캐나다 {매장명}" / 일반 유통 → "캐나다" / 50자 빠듯 → 출처 태그 생략.
- **"정품" 단어는 디폴트로 사용하지 않음.** 사용자가 명시적으로 "정품 넣어줘" 한 경우만 예외.
- `product_info.json`의 `source_store` 필드는 사용자가 제공한 매입처(예: "Loblaws") 그대로 기록 — 상품명 표기와 별개.

## 매장 자동 가정 금지 — 검증 체크리스트 (2026-05-17 강화)

사용자가 **"코스트코 라인"**, **"코스트코 매대"**, **"Costco에서 샀음"** 같은 모호한 표현을 써도 그것만으로 **매장 한정 SKU로 가정하지 않는다.** 일반 유통 SKU를 셀러가 코스트코에서 사 와도 매장 한정은 아님. 다음 체크리스트로 분류:

| 신호 | 매장 한정 SKU | 일반 유통 SKU |
| --- | --- | --- |
| 브랜드 | Kirkland(PB), 매장 PB, 매장 콜라보 | 일반 NB(Kellogg's, Kraft 등 자체 브랜드) |
| 사이즈/구성 | Costco bulk(750g+, 멀티팩, 기프트세트) | 소매 단일 박스(333g, 일반 grocery 사이즈) |
| 라벨 표시 | "Only at Costco", PB 로고, 매장 콜라보 표기 | 매장 표시 없음 |
| 사용자 발언 | "코스트코 한정", "Costco-exclusive", "여기서만 팔아" | "코스트코에서 샀음", "코스트코 라인"(모호) |

**셋 다 매장 한정 신호면 매장명 태그. 하나라도 일반 유통 신호면 일반 유통으로 가정 (`캐나다` 단독).** 사용자 발언만으로는 매장 한정 단언 부족 — 브랜드·사이즈 신호가 일반 유통이면 사용자에게 1줄로 확인 권장.

**2026-05-17 Kellogg's Frosted Flakes Cookies & Crème 333g 학습:** 사용자가 "코스트코 라인" 표현 → 자동으로 "캐나다 코스트코" 태그 → 사용자 정정 "무조건 모든 제품이 다 캐나다 코스트코라고 assume하지마". Frosted Flakes 333g는 소매 단일 박스 + Kellogg's NB + Costco-only 표기 없음 → 일반 유통으로 분류해야 했음. 다음부터 NB + 소매 사이즈는 디폴트 "캐나다" 단독.
