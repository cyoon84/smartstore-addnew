스마트스토어 상품명·상세페이지에 외래 브랜드명을 한글로 표기할 때, **네이버에서 검색량이 더 많은 표기**를 우선한다. 음운적·표준 외래어 표기법보다 검색 노출이 더 중요.

**확정된 매핑:**
- Chosen Foods → **초슨푸드** (NOT 초즌푸드)
- Seaside Spa (Downy 향) → **씨사이드 스파** (NOT 시사이드스파, 띄어쓰기 있음)
- Webber Naturals → **웨버내추럴스** (사용자 확정)
- Digestive (영양제 라인 / 일반명) → **다이제스티브** (NOT 디제스티브 — 사용자 명시 정정 2026-05-15)
- Digestive Enzymes (제품 카테고리 일반명) → **소화효소** (NOT 단독 "효소" — 사용자 정정 2026-05-15, 소화효소가 검색량·명확성 모두 우세)
- Smucker's → **스머커스** (NOT 스머커즈 — 2026-05-24, 코스트코 코리아 공식 표기·국내 listing 통용)
- Goober (Smucker's PB+잼 스월 제품) → **구버** (한국 마니아·직구 유저 통용 표기)
- RW Garcia → **RW가르시아** (NOT 알더블유가르시아 — 영문 이니셜은 음역하지 말고 그대로 노출. 네이버쇼핑 검색 통용. 2026-06-21) · 제품 표준명 `RW가르시아 고구마 크래커`(`3씨드` 라인명은 한글 상품명에서 제외)

> 일반화: 영문 이니셜/약자 브랜드(RW·PC·H&S 등)는 음역(`알더블유…`) 금지 — 영문 약자를 그대로 한글 상품명에 노출하거나 검색 통용 표기를 따른다. 음역 풀이는 검색량이 거의 없음.

**Why:** 사용자가 스마트스토어 셀러라서 한글 키워드 검색 노출이 매출 직결. 표준 표기보다 실제 소비자가 검색창에 치는 단어를 따라가야 함.

**How to apply:**
- 새 브랜드 만날 때마다 두 가지 표기 후보가 있으면 네이버 검색량(또는 사용자 확인)으로 결정
- 결정된 표기는 이 파일에 누적해서 표기 사전화
- product_info.json의 `brand` 필드는 영문(원문) 유지, 한글 표기는 `title_ko`와 description 본문에 사용
- 이미 만든 상품 페이지에서 표기가 다르면 한 번에 일괄 수정
