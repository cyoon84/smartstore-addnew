**목적:** §10(네이버 태그 통과/거부 사전)은 태그 입력화면의 **추정 검색량**이 근거였는데, 비즈어드바이저 실시간 보고서(`https://bizadvisor.naver.com/summary/realtime` → 마케팅분석 → "키워드별 유입수 Top 10")는 **우리 스토어에 실제로 들어온 검색어**라 더 강한 신호다. 검색량 추정치가 아니라 **실제 전환 행동**이므로, 여기서 반복 관찰되는 패턴은 §10 태그 축·§5 상품명 규칙에 우선 반영한다.

**방법:** 화면이 30분 창(최근 30분 유입)이라 30~45분 간격으로 스크린샷 확인(computer-use, Chrome read 권한 — 클릭·스크롤 불가라 사용자가 미리 검색어 섹션까지 스크롤해둬야 함). 반복되거나 특이한 검색어는 아래 관찰 로그에 누적, 패턴화되면 LEARNED_RULES §0-E·§10에 반영.

## 관찰 로그

### 2026-07-15 (수) 밤 — 첫 관찰
- **"브라이언부스터"** (모바일, 네이버가격비교-통합검색) — 다우니 언스토퍼블에 붙인 **"청소왕 브라이언"** 네이밍(상품명 접두)을 고객이 그대로 기억해 브랜드명처럼 검색. 우리 자체 네이밍 컨벤션이 실제로 재검색을 유발한다는 실증 신호 — 앞으로도 캐릭터/닉네임형 상품명 접두를 유지할 근거.
- **"일본제bandaid"** (모바일) — `<원산지>+제+카테고리` 패턴. 기존 학습 원산지 축(`캐나다직수입`·`캐나다산`·`미국산`, §10)과 다른 신규 조합 `~제`. 일본산 밴드에이드류(7/10 정산 COGS "밴드에이드 220입") 관련 검색으로 추정 — 향후 이 SKU 태그·상품명에 `일본제` 접두 시도 가치.
- **"룰루레몬everywhere벨트백"** (모바일) — 브랜드+라인명(Everywhere)+카테고리(벨트백) 통짜 검색. 룰루레몬 벨트백 계열 SKU 상품명·태그에 참고.
- 유입채널 상위: 네이버페이 / 네이버플러스스토어-검색 / 네이버가격비교-카탈로그(검색) / 네이버가격비교-통합검색.
