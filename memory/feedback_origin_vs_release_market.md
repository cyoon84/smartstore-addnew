상품이 캐나다에서만 파는 한정품이어도 **제조국(원산지)은 캐나다가 아닐 수 있다.** 세라믹 머그·잡화·굿즈류는 통상 중국 등 해외에서 제조된다. 스토어 페이지에 제조국(country of manufacture)이 명시돼 있지 않으면 detail.html / product_info.json / 등록정보.md에 "원산지: 캐나다"라고 쓰지 말 것. 캐나다는 "출시·판매 시장(release market)"으로만 표기하고, 제조국은 미상으로 둔다. 네이버 원산지 필드는 사용자가 실물 라벨('Made in ___')을 확인해 입력하도록 안내.

**Why:** 2026-05-21 팀홀튼 토론토 16oz 세라믹 머그 등록 시 Claude가 "원산지: 캐나다"로 적었고, 사용자가 "원산지는 캐나다산 아닐 것"이라 정정. 캐나다 한정 출시 = 판매 시장이지 제조국이 아님. 검증 안 된 필드를 지어내지 않는다는 §9 데이터 진실성 원칙과 직결.

**How to apply:** 원산지/제조국은 스토어 페이지나 라벨에 명시된 경우에만 기재. 미명시면 "제조국 미상"으로 두고, 캐나다 한정 출시 사실은 release_market 으로 별도 표기. "캐나다직수입" 같은 태그는 구매·수입 경로를 뜻하므로 허용되지만, 제조국 표기로 오인될 표현(원산지 캐나다, 캐나다산)은 금지. 관련: [[feedback_title_source_tag]].

**🔑 보강 — "캐나다 직수입"도 고객 노출 카피에선 원산지 오해 주의 (2026-06-27):** `캐나다 직수입`은 의미상 "캐나다에서 직수입(경로)"이지만, **소비자는 '캐나다산(원산지)'으로 오해**할 수 있다. 그래서 **제품 원산지가 캐나다가 아니면(예: Downy=미국 제조, Hershey=미국 등) 고객 노출 카피(SNS 카드·상세 헤드라인·강조 배지)에서 "캐나다 직수입"으로 원산지를 암시하지 말 것.** 대안: `해외직수입`, 실제 원산지(`미국 정품`), 또는 매장 출처(`캐나다 매장 직구`)처럼 원산지를 단정하지 않는 표현. 원산지가 실제 캐나다인 제품(PC 메이플시럽·RW가르시아 등)엔 `캐나다 직수입` OK. (네이버 검색 태그로서의 `캐나다직수입`은 검색어라 별개지만, 원산지가 캐나다 아니면 카드·배지 등 강조 위치엔 넣지 않는다.) 사용자: "캐나다 직수입이라 하면 어떤 사람들은 진짜 캐나다산으로 헷갈려할 수 있어." [[feedback_naver_tags]]
