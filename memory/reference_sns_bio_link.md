**인스타그램(및 SNS) 프로필 바이오 링크:** https://linktr.ee/finchmart_ca (Linktree, 2026-06-27 사용자 설정)

- 인스타 캡션/카드의 **"프로필 링크에서 구매 👆"** CTA가 이 링크트리를 가리킨다 (캡션 URL은 클릭 안 되므로 바이오 링크로 유도하는 게 정석 — [[feedback_sns_instagram_cards]] / 카드뉴스 CTA).
- 스토어 본주소: `https://smartstore.naver.com/finchmart_ca` (개별 상품은 `/products/{id}`, [[reference_smartstore_id]]).
- 스토리에는 링크 스티커로 개별 상품 URL 직접 연결 가능. 피드(캡션) → 바이오 링크트리 / 스토리 → 직접 링크.
- 캡션 작성 시: 전체 URL을 본문에 길게 적지 말고 "프로필 링크" 또는 "네이버에 캐나다 핀치마트 검색"으로 유도.
