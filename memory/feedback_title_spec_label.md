스마트스토어 상품명에 수치 사양(단백질량·용량·수량 등)을 넣을 때, **단독 숫자/단위만 쓰지 말고 그게 무엇의 수치인지 라벨을 붙여 결합**한다.

예: 단백질 23g 제품 → 상품명에 `23g`만 쓰면 중량으로 오인됨 → **`단백질23g`** 으로 결합 표기.

**Why:** 2026-05-24 Flourish 단백질 팬케이크 믹스 케이스 — 50자 맞추려 '프로틴'을 빼면서 `1kg ... 23g`로 두니 23g이 무엇의 수치인지 모호(이미 1kg 중량이 있어 더 헷갈림). 사용자 지적 "23g는 프로틴(단백질)인데".
**How to apply:** 상품명에 스펙 숫자를 넣을 때 항상 `단백질23g`, `용량1kg`처럼 라벨+수치 결합형으로. 50자 제한에 걸리면 중복 단어(같은 뜻 '단백질'/'프로틴' 동시 노출 등)를 먼저 줄여 공간을 확보. 관련: [[feedback-title-source-tag]].
