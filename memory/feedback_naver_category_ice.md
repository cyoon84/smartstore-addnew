네이버 스마트스토어 카테고리에 **"빙과/아이스 > 슬러시"** 노드는 존재하지 않는다. 슬러시·아이스바·얼려먹는 빙과(Kisko Freezies 등 DIY-freeze 슬러시 포함) 모두 다음 카테고리로 등록한다.

**확정 카테고리:**
```
식품 > 아이스크림/빙수 > 아이스크림
```

**Why:** 2026-05-18 Dr Pepper Freezies 48팩 등록 시 1차 제안한 "식품 > 빙과/아이스 > 슬러시"는 네이버에 존재하지 않는 카테고리. 사용자 정정으로 "식품 > 아이스크림/빙수 > 아이스크림" 확정. 네이버 분류 체계는 빙과·슬러시·아이스바를 모두 "아이스크림" 하위로 묶음.

**How to apply:**
- 슬러시(slush) / 아이스바(ice bar) / 얼려먹는 빙과 / freezies / freeze pops / popsicle 류 신상 등록 시 자동 적용
- 카테고리 후보 제시 단계에서 "빙과/아이스 > 슬러시" 같은 가공된 후보 만들지 말 것 — 곧바로 "식품 > 아이스크림/빙수 > 아이스크림" 단일 후보로 진행
- 단, 빙수(팥빙수·과일빙수)는 같은 상위 노드 아래 "빙수" 분기가 따로 있을 수 있으니 빙수 SKU는 사용자 확인 필요
