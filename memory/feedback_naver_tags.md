**추가 정정 — 검색 효과 / 차단 패턴 (2026-05-24 Hershey's S'mores 키트 케이스 학습):**

사용자가 등록 시 실제로 쓴/버린 태그를 알려줌. 정책 통과 여부뿐 아니라 **실제 검색 트래픽**까지 봐야 함:

- 🚫 **서브브랜드/변형 브랜드명도 차단** — `리세스`(Reese's) 같은 변형/서브브랜드명도 브랜드라 ❌. 모브랜드뿐 아니라 라인업 내 변형 브랜드명까지 금지. (과거 Nestlé 케이스에서 킷캣·스마티스를 변형 태그로 쓴 건 잘못 — superseded.) 단 `쿠키앤크림`처럼 **맛/유형 서술어**는 브랜드가 아니므로 ✅.

**추가 정정 — '적용 안 되는 태그' (2026-05-24 Flourish 단백질 팬케이크 믹스 케이스 학습):**
