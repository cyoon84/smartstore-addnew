병행수입(정품 사입) 리셀러도 국내 독점수입권자가 위조상품으로 오인해 상표권 신고를 하면 조사·판매중지 대상이 될 수 있다. 정품 사입이라 원칙적으로 안전하지만 즉시 대응할 수 있게 준비해둔다.

**Why:** 2026-07-09 등록정책 스터디에서 확인 — kpaanews.or.kr 병행수입 상표권 신고 사례, silmupack.com 구매대행 등록가이드. 대부분(구매대행 통관분류·PCCC·업종코드 525105)은 우리가 사입(재고보유, 국내발송) 모드라 비해당임을 재확인.

**How to apply:**
- 사입 건별 구매 영수증·매장 현지 사진을 보관(재고관리 엑셀·[[project_inventory_list]]이 사실상 이 역할) — 상표권 신고 대응 시 정품 증빙으로 제출.
- 상세페이지 이미지는 브랜드 공식 이미지 무단 도용 대신 사용자 실사진/호스팅 URL만 사용 — 이미 기존 워크플로가 이 원칙(§9 데이터 진실성)을 따르고 있어 별도 변경 불필요, 재확인만.
- "2026-06~ 판매실적 기반 상품등록 한도" 설(구매대행/대량등록 억제 목적)은 출처가 비공식(Threads 글, 수치 미확인)이라 반영 보류 — 다음 스터디에서 네이버 공식 공지 확인 후 재판단. [[feedback_ai_search_2026_bulk_penalty]] 방향과 일치하니 주시.
