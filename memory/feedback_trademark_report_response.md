병행수입(정품 사입) 리셀러도 국내 독점수입권자가 위조상품으로 오인해 상표권 신고를 하면 조사·판매중지 대상이 될 수 있다. 정품 사입이라 원칙적으로 안전하지만 즉시 대응할 수 있게 준비해둔다.

**Why:** 2026-07-09 등록정책 스터디에서 확인 — kpaanews.or.kr 병행수입 상표권 신고 사례, silmupack.com 구매대행 등록가이드. 대부분(구매대행 통관분류·PCCC·업종코드 525105)은 우리가 사입(재고보유, 국내발송) 모드라 비해당임을 재확인.

**How to apply:**
- 사입 건별 구매 영수증·매장 현지 사진을 보관(재고관리 엑셀·[[project_inventory_list]]이 사실상 이 역할) — 상표권 신고 대응 시 정품 증빙으로 제출.
- 상세페이지 이미지는 브랜드 공식 이미지 무단 도용 대신 사용자 실사진/호스팅 URL만 사용 — 이미 기존 워크플로가 이 원칙(§9 데이터 진실성)을 따르고 있어 별도 변경 불필요, 재확인만.
- "2026-06~ 판매실적 기반 상품등록 한도" 설(구매대행/대량등록 억제 목적)은 출처가 비공식(Threads 글, 수치 미확인)이라 반영 보류 — 다음 스터디에서 네이버 공식 공지 확인 후 재판단. [[feedback_ai_search_2026_bulk_penalty]] 방향과 일치하니 주시.

**추가 학습 (2026-07-13, 후속 스터디):**
- 정품 증빙(영수증·매장사진)만으론 100% 방어 안 되는 케이스 존재 — 국내 업체가 특정 브랜드의 **"전용사용권"을 단독 설정**한 경우, 진정상품이어도 병행수입 유통 자체가 위법 판단될 수 있음(비공식 법률블로그 종합, 신뢰도 중). **추가 방어책:** 신규 브랜드 취급 전 KIPRIS(kipris.or.kr)에서 그 브랜드 상표의 전용사용권 설정 여부를 사전 확인하는 습관 권장(제안 — 워크플로 강제화는 보류, 상표권 신고 들어왔을 때 1차 대응으로 KIPRIS 조회).
- 2025-07-22부터 **악의적 상표권 침해 징벌적 손해배상 한도가 3배→5배로 상향**(공식, 상표법 개정) — 신고 시 리스크 자체가 커졌다는 배경으로 참고. 경고장 수령 시 즉시 합의금 지급 의무는 없고, 등록 유효성(KIPRIS) 확인이 1차 대응.
- 전자상거래법 "국내대리인 지정 의무화" 개정(2026-01-20 공포)은 **시행이 2027-01-21**이라 지금은 무관 — 캘린더 재확인 대상으로만 표시(대상도 "해외 사업자" 자체라 국내사업자인 우리가 적용받을지도 불확실).
- 사입 모델 관련 기존 이해(업종코드 525101, 과세기준=전체 판매가, PCCC 구매대행 전용, 관부가세는 우리가 흡수해 등록가에 녹이는 방식)는 새 소스로도 재확인됨 — 변경 없음.
