네이버 스마트스토어에서 그룹 상품(여러 옵션을 묶은 상품)을 한 번 만들고 나면, **그룹 상품명과 기존 옵션명을 사후에 수정할 수 없다.** 새 옵션을 추가하는 것만 가능.

**Why:** 사용자가 실제로 "다우니 언스타퍼블 향기부스터" 그룹을 만든 뒤 "다우니 향기부스터"로 일반화하려 시도했으나 수정 막혀서 불가능했음. 그래서 비-Unstopables 라인 제품(2in1 Soft+Fresh)도 기존 그룹에 들어가야 함.

**How to apply:**
1. **그룹 상품을 처음 만들 때부터 가장 포괄적인 이름으로 시작.** 예: "다우니 언스타퍼블 향기부스터" 대신 "다우니 향기부스터", "OO 향 캔디 12개입" 대신 "OO 캔디 12개입" 같은 패턴.
2. **옵션명 컨벤션도 처음부터 통일.** 향/맛/사이즈 등 변형 차원이 명확히 드러나는 패턴을 첫 옵션부터 적용.
3. 새 라인 제품을 기존 그룹에 추가할 때, 그룹명이 라인 특정적이면 (A) 그대로 옵션명에 라인 구분 박아서 넣기 / (B) 별도 그룹 신설 두 옵션 제시. 기본 권장은 A — 기존 그룹 트래픽 활용이 더 큰 이득.
4. 사용자가 그룹 상품을 만들겠다고 하면, 등록 직전에 "그룹명/옵션명 컨벤션 이대로 확정해도 될까요? 나중에 수정 불가합니다" 확인.
