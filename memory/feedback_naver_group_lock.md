네이버 스마트스토어 그룹 상품(여러 옵션을 묶은 상품)에서, **그룹명(상품명)도 기존 옵션도 생성 후 수정 가능**하다. 새 옵션 추가도 가능. (2026-06-21 사용자 정정 — 이전엔 "그룹명만 수정 가능, 기존 옵션은 잠겨 수정 불가"로 알고 있었으나 그것도 오류. 옵션도 수정 가능.)

**Why:** 2026-06-21 룰루레몬 패스트앤프리 러닝 벨트백(12514179842) 그룹 전환 상담 중, 내가 "기존 옵션은 잠겨 수정 불가"라고 안내하자 사용자가 **"정정 — 수정 가능"**이라고 바로잡음. 즉 그룹명·옵션 둘 다 사후에 고칠 수 있다. (2026-06-09엔 "옵션 잠김"으로 알았으나 superseded.)

**How to apply:**
1. **그룹명·기존 옵션값(맛·용량·색상) 모두 사후 수정 가능** — 잘못 만들어도 나중에 고칠 수 있으니 옵션 생성 단계에서 과도하게 겁먹지 말 것. 단 불필요한 수정은 SEO 흔들림·혼선 유발하니 처음부터 잘 잡는 게 베스트.
2. **그룹명이 새 옵션과 안 맞으면(예: "코스트코 대용량"인데 소용량 추가) 그룹명을 일반화해서 수정**한다.
3. 그룹명엔 옵션값(맛·용량·색상) 넣지 않음 — 네이버가 옵션 별도 결합 노출(§13-1). first-option 포함 룰은 "단일상품+옵션여럿"(§13-4)에만.
4. 그룹명+옵션 결합 문자열에 **반복 단어 다수면 거부** — 새 옵션/그룹명 지을 때 조합 후 반복어 점검(§13-1).
