네이버 스마트스토어 그룹 상품(여러 옵션을 묶은 상품)에서, **그룹명(상품명)은 생성 후에도 수정 가능**하지만 **기존 옵션은 잠겨서 수정 불가**다. 새 옵션을 추가하는 것은 가능. (2026-06-09 사용자 정정 — 이전엔 "그룹명도 잠긴다"고 잘못 알고 있었음.)

**Why:** 2026-06-09 Awake 카페인 초콜릿 그룹(50824648)에 8개입(108g) 옵션 추가 작업 중, 그룹명 `Awake 카페인 초콜릿 캐나다 간식 코스트코 대용량` 이 8개입엔 안 맞아 "코스트코 대용량" 제거가 필요했는데 — 사용자가 **"그룹명은 수정 가능, 옵션이 잠겨서 수정 불가"**라고 정정. 즉 그룹명이 라인/용량 특정적이어도 **나중에 그룹명을 고쳐 일반화할 수 있다.**

**How to apply:**
1. **기존 옵션값(맛·용량·색상)은 사후 수정 불가** — 새 옵션을 추가할 땐 기존 옵션 컨벤션에 맞춰 새 옵션값만 넣는다.
2. **그룹명이 새 옵션과 안 맞으면(예: "코스트코 대용량"인데 소용량 추가) 그룹명을 일반화해서 수정**한다. 그룹명은 풀 수 있으니 막지 말 것.
3. 그래도 처음부터 포괄적 그룹명 권장(불필요한 수정·SEO 흔들림 방지). 그룹명엔 옵션값(맛·용량·색상) 넣지 않음 — 네이버가 옵션 별도 결합 노출(§13-1). first-option 포함 룰은 "단일상품+옵션여럿"(§13-4)에만.
4. 그룹명+옵션 결합 문자열에 **반복 단어 다수면 거부** — 새 옵션/그룹명 지을 때 조합 후 반복어 점검(§13-1).

(이전 "다우니 향기부스터" 케이스에서 그룹명 일반화가 막혔다고 기록돼 있었으나, 실제로는 그룹명 수정 가능 — 당시 다른 제약이거나 오진단으로 보임. 현재 기준: 그룹명 O / 옵션 X.)
