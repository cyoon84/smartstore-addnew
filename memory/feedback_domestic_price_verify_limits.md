§0-A 국내 시세 조사(market-researcher/웹검색) 결과는 **stale·사장(死藏) 리스팅을 포함할 수 있다.** 검색으로 잡힌 저가 리스팅을 "현재 살아있는 최저가"로 단정하지 말 것.

**Why:** 2026-07-02 허니번치스 오브 오츠 아몬드 케이스 — market-researcher가 11번가 ₩13,500·위메프 ₩9,700 등을 근거로 "가격으로 안 됨" 판정했으나, 사용자가 11번가 링크를 직접 클릭하니 **"판매되지 않는 상품"**(사장)이었다. 저가 근거가 stale 이었고 판정을 철회함.

**How to apply:**
- **한국 커머스 사이트는 Chrome MCP 로 직접 검증 불가** — 네이버쇼핑·쿠팡·11번가·위메프 모두 "안전 제한(This site is not allowed)"으로 차단. WebFetch 도 JS 렌더라 껍데기만 옴(가격·판매상태 안 보임). 즉 내 도구로는 한국 마켓 실시간가 확정이 사실상 어렵다.
- §0-A 보고 시 **"검색 기준·현재 판매 여부 미확인"**을 명시하고, 저가 리스팅은 "확인 필요"로 표기. "가격으로 안 됨"을 강하게 단정하기 전에 링크 생존을 의심.
- 사용자가 "링크가 죽었다/판매 안 된다"고 하면 즉시 판정을 완화·철회하고, **네이버쇼핑에서 직접 검색해 현재 셀러·최저가 확인**을 사용자에게 요청(내가 못 여는 것을 인정).
- 등록 진행(Go) 시 등록정보 §0-A 란에 "불확실 — 실경쟁가 모니터링 권장" 기록. [[feedback_domestic_price_check]]

> 관련: 아이보리·뉴트로지나·허니번치스 등 코스트코/월마트 대용량 그로서리는 국내 직수입이 촘촘한 편이나, 개별 SKU 의 현재 생존 리스팅은 매번 stale 위험이 있으니 검색 결과를 과신하지 말 것.
