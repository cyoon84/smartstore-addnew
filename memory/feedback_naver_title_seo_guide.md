2026-05-24 사용자가 네이버 공식 "쇼핑검색 SEO & 상품정보 제공 가이드"(NAVER, 2018-10-18) PDF를 제공하며 **"다음부터 이걸 반영해 상품명 작성"** 지시. 신규 등록 상품명은 아래 규칙을 따른다.

## 권장 구성 필드 (있으면 이 순서대로)
브랜드/제조사 → 시리즈 → 모델명(모델코드) → 상품 유형(형태) → 색상 → 소재 → 패키지 내용물 수량 → 사이즈 → 성별·나이 → 속성(Spec·용량·무게·연식·호수).

## 핵심 규칙
- **50자 내외 권장.** 50자 이상은 어뷰즈 판단 확률↑ (최대 100자). → 실무상 **45자 안팎, 최대 50자 미만**을 목표로.
- **조사·수식어 사용 안 함.** 공식 상품 정보만.
- **동의어/유의어 반복 금지** — 네이버가 동의어를 자동 처리하므로 중복 기재 불필요. 예: `단백질`과 `프로틴`을 둘 다 넣지 말 것 → **하나만**(한글 외래어 표기 우선).
- **브랜드·제조사·동일 단어 반복 금지.**
- **셀러·쇼핑몰명·상호명·판매처명 상품명에 포함 금지** (판매처는 별도 노출됨). ⚠️ **매장명(코스트코·로블로스 등)도 판매처명** — BAD 예시 "[롯데백화점][롯데일산점]". → [[feedback-title-source-tag]] 와 충돌, 아래 참조.
- **한글 우선, 필요 시에만 영문.** 외래어 브랜드는 한글 표기 우선(예: Flourish → 플로리시). 한글/영문 외 언어·한자 금지. 숫자는 아라비아.
- **허용 특수문자: ( ) - · [ ] / & + , ~ . 만.** 그 외 기호 금지, 특수문자도 과다 사용 X.
- **이벤트·할인·배송·쿠폰·적립·판매조건 기입 금지** (별도 필드).
- **홍보성 수식어 금지 단어:** 정품·공식·한정·인기·최고·가성비·추천·신상품·특가·최저가·즉시할인·재입고·MD추천·1위·선착순 등.
- **카테고리를 정확히 매칭했으면 상품명에 카테고리 키워드를 불필요하게 넣지 않음** (카테고리명 관련 키워드는 카테고리만으로도 검색됨).
- 패키지 상품은 내용물 수량 정확히. 하나의 상품·단일 정보만(서로 다른 상품 묶기 X).
- "정품" 표시는 지양(상품상세에서 안내). 시즌 표시는 "연도+계절".

- 아모레퍼시픽 헤라 옴므 셀 프로텍션 선커버 레포츠 50ml (SPF50)
- 마미포코 팬티형 기저귀 4단계 대형 38매 (남여선택)

## 기존 룰과의 충돌 — 해소됨 (2026-05-24)
본 가이드는 매장명(판매처명)을 상품명에 넣는 것을 금지하나, `[[feedback-title-source-tag]]` 의 매장 한정 SKU "캐나다 코스트코" 출처 태그에 대해 **사용자가 "캐나다 코스트코는 유지하자"고 명시 결정**. → 매장명 출처 태그는 예외로 유지하고, 본 가이드의 나머지 규칙(동의어 중복 금지·50자 내외·홍보성 단어 금지·한글 브랜드 표기·특수문자 제한 등)은 전부 적용. 이 1건만 가이드 예외.

## 태그 (가이드 §3.7 — [[feedback-naver-tags]] 보강)
태그 사전 미등록 태그는 노출 안 됨. 감성 표현 태그 권장. 브랜드·제조사·판매처는 태그 중복 금지(각 필드에 입력).
