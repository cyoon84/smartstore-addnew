상품명 끝에 "캐나다 코스트코" 같은 출처/매장 태그를 붙이면, 상세페이지 본문(헤드라인 후크, 소개, 포인트, 추천 대상, 마무리 등)에서는 같은 표현을 반복하지 않는다.

**Why:** 상품명에서 이미 검색 키워드/신뢰 시그널을 잡았기 때문에 본문에서 반복하면 카피가 장황하고 제품 자체 매력보다 출처 강조에 시선이 분산됨. 사용자는 본문이 제품 자체에 집중되길 원함.

**How to apply:** 상세페이지 HTML 작성 시 상품명에 들어간 출처 태그(예: "캐나다 코스트코", "캐나다 정품", "코스트코" 등)는 본문에서 빼고 제품 특성(맛/용량/조합/타겟 사용 상황) 중심으로 작성. 브랜드의 국가 라인 언급(예: "Kraft Canada 한정 라인")은 OK — 출처 태그 반복과는 다른 정보.
