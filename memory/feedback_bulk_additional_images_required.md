네이버 스마트스토어 **일괄등록 엑셀의 추가이미지(X)는 필수가 아니다.** 여러 이미지가 확보돼 있으면 넣어서 올리고, **대표 이미지 하나뿐이면 추가이미지는 그냥 안 넣어도 된다** (대표이미지 W만 있으면 등록 가능).

**Why:** 2026-07-05 ATTITUDE 키즈 샴푸 등록 때 "삐삑! 추가이미지 필드 없다"를 보고 한때 "추가이미지=필수"로 규칙화했으나, 사용자 정정: **필수 아님.** ("추가이미지 필수는 아니지만 있으면 올리고, 만일 이미지가 하나면 걍 안 해도 돼.")

**How to apply:**
- 상세/갤러리 이미지 URL 이 여러 개 있으면 `bulk.add_images`(list/str, 최대 9개, 줄바꿈 구분) 또는 `images.additional_image_urls` 에 넣어 X 칸을 채운다(대표와 중복 X, 나머지 제품컷·인포그래픽; 호스팅 Flickr 등).
- **이미지가 대표(rep) 하나뿐이면 추가이미지는 비워도 정상** — 억지로 채우려 하지 말 것.
- bulk-excel-verifier 필수체크에서 **추가이미지는 FAIL 사유 아님**(대표이미지만 필수). 추가이미지 유무는 정보성 표기로만.
