핀치마트가 취급하는 P&G 아기 세탁 라인(아기세제·블리스풀 향기부스터 등)의 **네이버 등록 브랜드명은 `아이보리`(영문 `IVORY`)** 다. `아이보리스노우`가 아니다 (사용자 확정 2026-07-14 — "브랜드 IVORY야, 아이보리스노우 아니고. 내가 다 고쳤어").

- 상품명·브랜드 필드·상세 카피 = **아이보리** / **IVORY** 로 통일.
- 기존 핀치마트 아기세제 리스팅도 브랜드 `아이보리`로 등록돼 있음 → 신규 P&G Ivory 계열(블리스풀 향기부스터 등)도 **아이보리**로 맞춰 크로스셀 일관성 유지.
- 예외: 영어 **모델명** 필드(§5 영어 공식 풀네임)는 봉지 인쇄 그대로 `Ivory Snow Blissfuls…` 유지 가능(로마자 검색 매칭·§9 사실). 단 한글 브랜드/상품명엔 "스노우" 빼고 `아이보리`.
