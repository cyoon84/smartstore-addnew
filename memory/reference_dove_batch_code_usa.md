핀치마트가 취급하는 **도브(Dove)** 제품은 **미국 공장 제조 정품**이다 (소싱은 캐나다 코스트코지만 제조국은 미국). 고객이 "이거 미국산 맞냐"를 자주 물어서, 상세페이지에 **배치코드로 미국 제조를 증명**하면 신뢰·전환이 오른다 (사용자 지시 2026-07-14).

**배치코드 해독 (포장 각인):** 예 `03135XU04`
- `0313` = 제조 **3월 13일**
- `5` = **2025년** (연도 끝자리)
- `XU04` = **미국 공장 코드**
- 유통기한 = 제조 후 **36개월** (화장품 일반)
- 참조: blog.naver.com/jewel2248/223445667210 (도브 배치코드 해독)

**원산지 = 미국 (0204000).** 소싱국(캐나다 코스트코)이 아니라 실 제조국(Made in USA) 기준. 기존 리스팅이 캐나다로 돼 있으면 미국으로 수정. [[feedback_brand_country_vs_made_in]]

**상세 적용:** "🇺🇸 미국 제조 정품 확인" 섹션 = 배치코드 각인 실사진 + 해독 설명 + FAQ("미국산 맞나요?" → 미국 공장 XU04 정품). 단 특정 제조일은 입고 배치마다 다르므로 "예시 코드"로 표기하고 "배치에 따라 다를 수 있음" 1줄. (§9 진실성)

첫 적용: 도브 센서티브 뷰티바 106g 16개입 (제품번호 10556673976, slug `dove_sensitive_beauty_bar_16`). 도브 센서티브 = ¼ 보습크림·무향(fragrance free)·저자극(hypoallergenic)·피부과 추천#1. 다른 도브(맨케어·오이그린티·바디스크럽 등)에도 배치코드 미국제조 증명 재사용.
