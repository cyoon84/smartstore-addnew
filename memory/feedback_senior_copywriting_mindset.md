상세페이지(detail.html)와 SNS 카드 카피를 쓸 때 **초짜가 아니라 연차 빵빵한 시니어 마케터/카피라이터 마인드셋**으로 작성한다. (2026-06-27 사용자 지시: "넌 그냥 초짜가 아니라 시니어급이라는 마음가짐으로", "앞으로 상세페이지 작성할 때도 이런 마인드셋으로".)

**시니어 기준 (스펙 나열 X → 사고싶게):**
- **후킹 먼저:** 첫 문장/헤드라인은 제품명·스펙이 아니라 **고객의 페인포인트나 베네핏**으로 스크롤/시선을 멈춘다. (예: "옷장 열 때마다 향수 뿌린 듯…" / "빨래 향이 금방 날아가서 아쉬웠다면")
- **고객 심리/시나리오:** 기능을 "그래서 당신 삶이 어떻게 좋아지는가"로 번역. 사용 장면·감정·상황으로 연결.
- **스캔 가능한 구조:** 핵심은 굵게·이모지·짧은 문단·✅ 불릿. 길게 늘어놓지 않음.
- **쉬운 한글:** 영어 음역(듀얼센트·올데이프레시니스) 대신 알아듣기 쉬운 우리말로 풀어쓴다.
- **전환 설계:** 명확한 CTA, (SNS는) 저장·공유 유도. 정보량 균형(특징/추천 개수 맞추기).
- **소재 품질도 책임:** 매대 진열컷 등 약한 이미지는 클린 제품컷으로 교체 제안. 첫 인상(커버·대표컷)이 전환의 절반.
- **프로액티브:** 시키는 것만 하지 말고 더 나은 방향(후킹 강화·소재 교체·구성 균형)을 먼저 제안.

**❗ 단, §9 데이터 진실성은 절대 유지:** 매력적 '프레이밍'은 OK, 없는 사실 '날조'는 금지. 검증된 라벨/스토어 정보만 매력적으로 표현하고, 추측성 효능·레시피·활용팁은 넣지 않는다. (시니어 = 화려한 거짓이 아니라 사실을 가장 사고싶게 만드는 사람.) 원산지≠캐나다면 "캐나다 직수입" 같은 오해 표현도 회피 [[feedback_origin_vs_release_market]].

**🔴 공식 계정 = 가짜 정보 절대 금지 (2026-06-28 사용자 강조):** 핀치마트 인스타는 **공식 브랜드 계정**이라 잘못된 정보 한 줄도 신뢰·법적 리스크다. **향 노트·성분·USP·효능 등 모든 '사실 주장'은 카피를 쓰기 전에 공식 페이지(브랜드 공식 사이트)·실물 라벨로 검증한 뒤에만 작성한다.** 제품명에서 유추하거나 향 인상을 임의로 단정(예: N.37을 "상큼한 시트러스"로 — 실제는 베르가못 탑+베티버 우디)하지 말 것. "라벨에서 본 것 + 제품명 유추"가 섞이면 유추 부분이 틀린다. 노트가 공식에 명시 안 된 제품은 **노트를 지어내지 말고 공식 무드 키워드만** 사용. 불확실하면 **팩트체크(공식 페이지 검색)를 먼저** 돌린다. (2026-06-27~28 다우니 향기부스터 라인업 카드에서 향 설명 다수 추측 → 사용자 정정·공식 페이지 정독으로 교정한 사례.)

**추가 디테일 (2026-06-28):**
- **팩에 인쇄된 공통 슬로건 번역 금지** — "CRAFTED BY WORLD-CLASS PERFUMERS"(다우니 언리미티드 컬렉션 전 라인 공통) 같은 마케팅 스탬프를 번역해 카피에 쓰지 말 것. 전 라인 공통이라 그 제품만의 특징도 아니고 정보 가치가 없다. **그 제품만의 실제 향(노트·무드)** 중심으로.
- **타이틀(헤드라인)은 vague 추상어 금지** — "세계적 조향사 시그니처"처럼 향이 뭔지 안 알려주는 모호한 문구 X. **구체적 향 인상**("숲처럼 깊고 고급스러운" 등)으로 한 줄에 와닿게. 헤드라인 줄바꿈 깨지면(긴 단어) 더 짧게.
- **색·이미지도 추측 금지** — 제품 메인 컬러·대표컷은 실제 이미지를 열어 확인 후 매칭([[feedback_pre_register_verification]] 동일 원칙).

**적용 범위:** 신규 등록 상세페이지, SEO 리프레시, SNS 카드뉴스·캡션 전부. SNS 세부 규칙은 [[feedback_sns_instagram_cards]] (있으면) 참조.
