**Why:** 2026-06-26 하루에 반복 실수 — ① 언스토퍼블 N.26 대표이미지를 3개 사이즈 전부 `rep[0]`(598g) 한 장으로 깔음(이미지 안 열어봄) ② 브랜드·제조사를 `Unstopables`로 적음(다우니/P&G 미확인 — Unstopables는 P&G 다우니의 제품 라인) ③ 자미에슨 배송 "3개당 15000"을 개당 5000으로 자의 환산 ④ 자미에슨 콜라겐을 `기타건강보조식품(50002615)`로 강제(다이어트식품>콜라겐 선택지 미확인) ⑤ 맥카페/허쉬 태그 단정. 전부 사용자가 잡아줘야 했음.

**How to apply — 마무리 전 체크리스트:**
1. **이미지 (사이즈/맛 옵션 여러 개면 필수):** 대표이미지를 옵션마다 **한 장씩 실제로 열어 라벨(용량/맛)을 확인**하고 각 SKU에 맞게 매핑. `representative[0]` 일괄 사용 금지. 추가이미지 갤러리엔 타옵션 보틀 섞지 말고 사이즈 무관 상세컷 사용. (대표 N장 = 옵션 N개일 가능성 먼저 의심) [[feedback_bulk_upload_excel]]
2. **브랜드/제조사:** 제품 라인명(Unstopables·N.26 등)을 브랜드/제조사 칸에 넣지 않는다. **모브랜드**(예: 다우니)와 **실제 제조사**(예: P&G)를 확인해 brand=모브랜드, maker=제조사로. 형제 제품 product_info의 brand/maker와 대조. [[feedback_model_name_from_store]]
3. **배송 "M개당 N원":** 사용자 명시 문구는 **환산 없이 액면 그대로**(수량별 fee=N·AP=M). "= 개당 X" 분담 환산으로 바꾸지 말 것. [[feedback_shipping_per_unit_no_bundle]]
4. **카테고리:** 영양제처럼 보여도 강제코드(50002615) 전에 **더 맞는 세분류가 선택 가능한지 확인** — 콜라겐=식품>다이어트식품>콜라겐(50007037) 등. 형제 제품이 쓴 카테고리 대조. [[feedback_bulk_upload_excel]]
5. **태그:** 입력화면 검증 전 단정 금지. 거부 위험 후보(IP·카테고리어)도 `<IP>굿즈`처럼 결합형은 통과할 수 있으니 후보로 제시. [[feedback_naver_tags]]
6. **제품 유형 명칭은 지어내지 말 것:** 형제/기존 listing 이 쓰는 표준 표기를 따른다. 예 — 인워시 스캔트부스터를 `향기알갱이`(임의 조어)로 쓰지 말고 형제 다우니 파라다이스 표기 **`향기부스터 비즈`**(In-Wash Scent Booster Beads)로. 상품명·헤드라인 짓기 전 같은 제품군 형제 폴더의 title/detail 표기 먼저 대조. (2026-06-26 N.26 케이스 — 사용자 지적)
