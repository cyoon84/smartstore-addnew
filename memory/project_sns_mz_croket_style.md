**구현 완료 (2026-06-28).** `/sns-instagram` 의 2번째 모드 = **MZ 라인업/포토덤프 (croket 스타일)**. 사용자 레퍼런스 @croket_official.

**스크립트:** `scripts/build_sns_mz.py [그룹ID] [출력폴더명]` (기본 50533349=다우니 언스토퍼블 / downy_unstopables_lineup_mz)
- guide/Product_*.csv 에서 **그룹상품ID** 로 라인업 추출 → 사이즈 변형 향별 1장 dedupe(대표이미지는 **최대 용량 컷**) → 4:5(1080×1350) 카드 렌더(Chrome 헤드리스).
- **캐러셀:** 커버(+"※ 예고 없이 추가·단종" 디스클레이머) → 향별 카드 → 💝 향 추천 가이드(페르소나→향) → 🛒 마무리 CTA(네이버 초록창).
- content layer(`COPY`/`PICK`/cover/guide)는 그 라인업에 맞게 교체. 출력 `output/sns/<폴더명>/cardNN.png`.

**디자인 스펙 (croket):** 실사진 풀블리드 + 하단 어둠 그라데이션 / 마커 헤드라인 폰트 **BMDOHYEON(도현체)** + **노란(또는 제품 메인컬러) 형광펜 하이라이트** / 흰 말풍선(제품색 테두리) / **좌상단 핀치마트 로고 칩** / 하단 흰 설명 박스(텍스트만, 제품색 좌측 액센트) / 향별 **제품 메인 컬러 테마**(이미지 열어 확인) / 영상 노출 제품엔 **🧹 청소왕 브라이언 추천템** 리본([[reference_sns_bio_link]]도 참조).

**🔑 역할 파이프라인 (MZ → 시니어 마케터):**
1. **MZ** = 향별 포토 쇼케이스 카드 제작(이미지·마커카피).
2. **시니어 마케터** = MZ 결과물 **팩트체크**(공식 페이지 정독, [[feedback_senior_copywriting_mindset]]) → 뒤에 **향 추천 가이드 + 마무리 CTA 카드 추가** + **캡션** 작성.
3. 팩트체크에서 향 설명·이미지·컬러 문제 발견 시 **MZ 에게 수정 지시**(재작업).

**원칙:** 공식 계정이라 가짜정보 0 / 향 노트·무드는 공식 검증분만(팩 공통 슬로건 번역·vague 타이틀 금지) / 원산지≠캐나다면 캐나다직수입 배지 금지([[feedback_origin_vs_release_market]]) / 캡션은 바이오 링크트리·네이버검색 유도 + 댓글 키워드 + 해시태그.

> 단일 제품(라인업 아님) 카드는 시니어 브랜드 카드 모드 `scripts/build_sns_cards.py`(코랄/크림 4장: 커버·특징·추천·CTA). MZ 라인업은 `build_sns_mz.py`.
