사용자가 어떤 SKU를 **"검색 유입용"**, **"끼워팔기용"**, **"마진 포기/최소"** 라고 명시하면 그 상품은 traffic-driver(loss-leader)다. cost-plus 통상 마진($2~5) 대신 **마진을 최소(예: 1,000원 수준)** 로 잡는다. 마진 단위가 KRW로 명시될 수 있으니 단위 확인.

**Why:** 객단가를 올리는 끼워팔기 상품은 그 자체 수익이 아니라 스토어 검색 노출·유입이 목적. 마진을 키우면 가격 경쟁력이 떨어져 유입 효과가 사라진다. (2026-05-20 Nespresso 버츄오 오다치오 캡슐 케이스 — 한국에도 유통되는 제품, "검색 유입용이니 마진은 포기" 명시.)

**How to apply:**
- 마진 최소 모드여도 네이버 수수료 6.6% gross-up은 그대로 적용 권장 — 안 하면 진짜 손해. 단 가격 경쟁력이 핵심이므로 gross-up 적용본/미적용본 둘 다 제시하고 사용자가 고르게.
- 한국에도 유통되는 제품이면 §0-A 국내 시세 확인은 여전히 의미 있음(한국 가격과 경쟁해야 유입 효과). [[feedback-domestic-price-check]] 적용.
- 끼워팔기 전제이므로 단품 기준 가격 열세는 의도된 것 — §0-A "가격으로 안 됨" 판정이 나와도 자동 접지 말고 전략 맥락을 보고.
