**어린이제품 KC 인증 — 판매모델에 따라 다르다. 완구를 무조건 "인증대상 아님"으로 뭉뚱그리지 말 것.** (2026-07-09 KC 대행사 LooKC/아이씨티컴플라이언스 블로그로 정정.)

- **"어린이제품" 정의:** 만 13세 이하가 사용/위하여 사용되는 물품·부속품(초등학생까지). 사실상 어린이와 스치는 제품 전부 KC 인증 대상.
- **KC 인증 3종:** ①안전인증(공장심사 O — 물놀이/놀이기구/카시트/비비탄총) ②**안전확인**(공장심사 X — **완구**·유아섬유·스포츠보호·자전거·학용품·유모차 등) ③공급자적합성확인(유해도 낮음 — 가구·킥보드·장신구 + 개별기준 없는 것 전부). **완구(레고)는 "안전확인" 대상.**
- **일괄엑셀 한계:** 일괄등록 템플릿엔 KC 인증 필드가 없음(인증 칸은 '상품정보제공고시 인증허가사항' 하나뿐, KC ≠). → 완구/유아/블록/레고/인형/키즈/출산·육아 업로드 시 네이버가 인증정보 미입력이라 **자동 판매중지** → 네이버 상품편집에서 인증정보 채워야 판매중.

**How to apply (판매모델 구분):**
- **구매대행**(주문 후 개인 직구 대행): 구매자가 개인수입자 → KC 인증 의무 없음 → 네이버 인증 = **'인증대상 아님/구매대행'** 선택 가능.
- **직접 수입·사입 후 재고 판매(핀치마트 기본)**: **판매자가 KC 안전확인 인증 의무자** → '인증대상 아님'은 원칙상 부정확, 적발 시 **판매정지·과태료** 리스크. 소규모 직구셀러가 회색지대 운영은 하나 **리스크 인지하고 셀러가 최종 판단**.
- `build_bulk_excel.py` 가 완구/어린이 카테고리 감지 시 이 뉘앙스 경고 출력.

**Why:** 처음 "완구=인증대상 아님 선택"으로 뭉뚱그렸다가 블로그로 정정 — 완구는 KC(안전확인) 대상이 원칙이고, "인증대상 아님"은 구매대행 전제. 2026-07-08 레고 메시 43015(사입 재고)를 '인증대상 아님'으로 올린 건 구매대행 스탠스라 리스크 존재(사용자 인지). [[feedback_bulk_upload_excel]] · LEARNED §16
