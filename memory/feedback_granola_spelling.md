Quaker Harvest Crunch 그룹상품 등록 시 사용자가 그룹명을 "퀘이커 하베스트 크런치 그라놀라 시리얼"로 확정 → **그라놀라** 표기 사용.

**Why:** 네이버 검색량 기준 표기 누적 사전 ([[feedback_brand_ko_names]] 동일 원칙). 사용자 직접 그룹명에 "그라놀라" 사용 (2026-05-26 케이스).

**How to apply:**
- Quaker 라인업 (Harvest Crunch 등) → "그라놀라" 사용
- Kirkland × Nature's Path 등 다른 브랜드는 기존 표기 유지 (kirkland_np_ancient_grains_granola_1kg는 "그래놀라" 사용 중) — 충돌 시 사용자 확인
- 향후 granola 카테고리 신규 등록 시 사용자에게 표기 한 번 확인 권장
